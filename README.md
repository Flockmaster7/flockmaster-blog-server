## :rocket:  Flockmaster Blog 后端​ ​ :rocket:

### :book:  ​前言

用 Koa2 + Sequelize 开发的 Flockmaster blog 个人博客后端

### :wrench:  ​技术栈

- 框架：Koa2
- ORM：Sequelize
- 数据库：Mysql、Redis
- 语言：Typescript
- body 解析：Koa-body
- 定时任务：Node-schedule
- 登录鉴权：Jsonwebtoken
- 服务启动：Nodemon

### :airplane:  快速开始​

```shell
npm install
```

```shell
npm run dev
```
